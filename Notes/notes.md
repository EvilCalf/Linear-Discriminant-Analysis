### np.sum

* 不传参数就是所有元素的和

* 传入参数axis，axis即是要消去的维度

如下所示：

```python
a=array([[[1, 2, 3, 2],
        [1, 2, 3, 1],
        [2, 3, 4, 1]],
       [[1, 0, 2, 0],
        [2, 1, 2, 0],
        [2, 1, 1, 1]]])


a.sum(axis=0)
array([[2, 2, 5, 2],
       [3, 3, 5, 1],
       [4, 4, 5, 2]])

a.sum(axis=1)
array([[ 4,  7, 10,  4],
      [ 5,  2,  5,  1]])

a.sum(axis=2)
array([[ 8,  7, 10],
       [ 3,  5,  5]])
```

### np.argmax
